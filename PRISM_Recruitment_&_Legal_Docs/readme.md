# PRISM Recruitment & Legal Docs

### 📩 Recruitment Messages (4টা)
| Message | Target | Key Hook |
|---|---|---|
| **Universal** | যেকোনো candidate-কে initial cold outreach | Project vision + equity mention, details পরে |
| **Member B** | Frontend developer | Next.js 15, React Native, TikTok-style UI — specific stack দেখানো হয়েছে |
| **Member C** | Backend developer | "real money movement, real regulatory compliance" — seriousness বোঝানো হয়েছে |
| **Member D** | QA + Growth person | "dual role: protect the platform AND grow it" — uniqueness highlight |

প্রতিটা message-এ **NDA mention আছে** — details দেওয়ার আগে sign করাবে।

### 📄 PRISM_NDA.docx
10টা section — সব কিছু cover করা:
- Confidential Information-এর সংজ্ঞা (technical plan থেকে equity split পর্যন্ত)
- **Non-compete clause** — 12 মাস একই domain-এ competing project করতে পারবে না
- IP protection, remedies, Bangladesh law, arbitration
- Bangla-তে summary notes প্রতিটা section-এ

---

### 📄 PRISM_Vesting_Agreement.docx
13টা section — founder-grade:
- Role-specific equity table (40/21/21/18 breakdown)
- **Vesting schedule table** — 3-month cliff → monthly over 12 months
- **Active Contributor Standard** — কতটুকু কাজ না করলে vesting আটকে যাবে
- "For Cause" termination সংজ্ঞা (breach, দীর্ঘ non-performance)
- Revenue distribution waterfall (খরচ → 20% reserve → equity split)
- Anti-dilution clause investor আসলে কী হবে
- IP assignment — সব code PRISM-এর
- সাক্ষী signature block

---

> **Flow টা হবে:** Universal message পাঠাও → আগ্রহ দেখালে role-specific message → NDA sign → full plan share → কাজ শুরু করলে Vesting Agreement sign।
